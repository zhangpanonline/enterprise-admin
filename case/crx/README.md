# 自动挂单

## 目录结构

```
glink-aog-app/
├── public/                  # 静态资源
│   ├── icons/               # 扩展图标
│   ├── manifest.json        # 扩展清单文件
├── src/
│   ├── background/          # 后台脚本
│   ├── content/             # 内容脚本
│   ├── popup/               # 弹出页面
│   ├── options/             # 选项页面
│   ├── assets/              # 静态资源
│   ├── styles/              # 全局样式
│   ├── utils/               # 工具函数
│   ├── types/               # 类型定义
│   ├── App.vue              # 主应用组件
│   └── main.ts              # 应用入口
├── vite.config.ts           # Vite配置
└── package.json
```

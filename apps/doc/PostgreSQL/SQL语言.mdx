## 一、[创建、删除表](https://postgres.ac.cn/docs/current/tutorial-table.html)

#### 创建表

```sql
CREATE TABLE weather (
    city            varchar(80),
    temp_lo         int,           -- low temperature
    temp_hi         int,           -- high temperature
    prcp            real,          -- precipitation
    date            date
);
```
